## 🛠️ Troubleshooting

### The app does not open
Try these steps:

1. Check that the download finished.
2. Make sure you extracted the zip file if you downloaded one.
3. Right-click the app and choose Run as administrator.
4. Check whether Windows blocked the file.
5. Download it again from the project page.

### Windows shows a security prompt
Choose the option to run the file if you trust the source and you downloaded it from the project page.

### The app opens, then closes
Try restarting your PC and opening the app again. If that does not work, download the latest version from the link above.

### The workspace looks empty
Create a new project or open an existing saved file. Many desktop tools start with a blank home screen.

### Files do not load
Check the file type and make sure it matches what the app supports. Try a smaller file first to test the import path.
